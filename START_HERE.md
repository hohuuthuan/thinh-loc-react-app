## 🎨 Customization

### Update Content
Edit `src/utils/data.ts`:
```typescript
// Services, Projects, Testimonials, Blog posts, Partners
export const SERVICES = [...]
export const PROJECTS = [...]
export const TESTIMONIALS = [...]
```

### Change Colors
Edit `tailwind.config.js`:
```javascript
primary: {
  600: '#YOUR_COLOR', // Primary blue
}
```

### Replace Images
Update image URLs in components (use your own or Unsplash)
